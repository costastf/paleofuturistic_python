Paleofuturistic Python

The Python development workflow your past self had always hoped for is finally here.

This is a cruft template that generates a fully-scaffolded, enterprise-ready Python package — uv-managed, ruff-formatted, pytest-tested, properdocs-documented, with vendored CI tooling, SBOM generation, and optional Dependency Track integration.
